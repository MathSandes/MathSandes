## 📌 Featured Projects

- **[financial-transactions-service](https://github.com/MathSandes/financial-transactions-service)**  
  API de transações financeiras com Node.js, Fastify e Prisma (PostgreSQL), arquitetura em camadas, valores em centavos, idempotência via `Idempotency-Key`, testes (Vitest) e CI no GitHub Actions.

- **[audit-log-service](https://github.com/MathSandes/audit-log-service)**  
  Serviço de auditoria com Node.js, TypeScript, Fastify, TypeORM e PostgreSQL — registro e consulta de logs com filtros por ator, entidade e paginação.

- **[product-catalog-service](https://github.com/MathSandes/product-catalog-service)**  
  Monorepo de um catálogo com API + worker assíncrono (BullMQ/Redis), enriquecimento via API externa, front em React (Vite), Prisma, Docker Compose e documentação OpenAPI/Swagger.
